# 素材格式兼容性参考

> 基于 **Qt 6.11.1 + GStreamer 后端**，运行于 **Quickshell 0.3.0**

---

## 一、📷 图片格式（`Image` / `BorderImage` 组件）

| 格式 | 扩展名 | 支持 | 说明 |
|------|--------|------|------|
| **PNG** | `.png` | ✅ **强烈推荐** | 无损，支持透明通道，加载快 |
| **JPEG** | `.jpg` / `.jpeg` | ✅ 支持 | 有损，无透明通道，适合照片级背景 |
| **SVG** | `.svg` / `.svgz` | ✅ **支持** | **可无限缩放不失真**，适合图标/UI元素 |
| **WEBP** | `.webp` | ✅ 支持 | 比 PNG 更小，支持透明和动画（静态帧）|
| **BMP** | `.bmp` | ✅ 支持 | 不推荐，体积大 |
| **GIF** | `.gif` | ⚠️ **仅静态帧** | 不支持动画，只显示第一帧 |
| **AVIF** | `.avif` | ⚠️ 需插件 | 新一代格式，需系统支持 |
| **TIFF** | `.tiff` | ⚠️ 部分支持 | 不推荐用于 UI |

### 📌 SVG 特别说明

SVG **可以替换**以下位置的 PNG/JPEG：

```
原文件                           可用 SVG 替换？
─────────────────────────────────────────────────
Assets/p3r imgs/bg.png           ❌ 壁纸太大，SVG 渲染性能差
Assets/components/mainm.jpeg     ❌ 照片类不适合 SVG
Assets/components/char1.png      ❌ 同上
Assets/components/player.png     ✅ 可！对话框背景适合 SVG
Assets/iconpack/shutdown.png     ✅ 可！图标类 SVG 最佳
Assets/iconpack/restart.png      ✅ 可！
Assets/iconpack/logout.png       ✅ 可！
Assets/img/wave1~16.png          ❌ 逐帧动画不适合 SVG
Assets/components/AlbumCover-by-Squirrel-Modeller.svg  ✅ 本来就是 SVG
```

> **结论**：SVG 最适合**图标、UI 边框、装饰性矢量图**。照片和壁纸类保持 PNG/JPEG。

## 二、🎬 视频格式（`Video` / `MediaPlayer` 组件）

当前系统使用 **GStreamer** 后端，支持以下容器/编码：

| 容器 | 视频编码 | 扩展名 | 支持 | 说明 |
|------|---------|--------|------|------|
| **MP4** | **H.264** | `.mp4` | ✅ **最佳选择** | 硬件加速好，兼容性最佳 |
| MP4 | H.265/HEVC | `.mp4` | ✅ 支持 | 更小体积，但编码较慢 |
| **WebM** | **VP8/VP9** | `.webm` | ✅ **推荐** | 开源，无损质量可选 |
| **Matroska** | **任意** | `.mkv` | ✅ 支持 | 兼容但容器略重 |
| **Ogg** | **Theora** | `.ogv` / `.ogg` | ✅ 支持 | 开源 |
| AVI | 多种 | `.avi` | ✅ 支持 | 不推荐，容器老旧 |
| **MOV** | **H.264/ProRes** | `.mov` | ✅ 支持 | Apple 生态，兼容性好 |
| Flash Video | — | `.flv` | ❌ 不支持 | 已淘汰 |
| WMV | — | `.wmv` | ⚠️ 部分支持 | Linux 下支持有限 |
| **HTML** | — | `.html` | ❌ **不支持** | HTML 不是视频格式！|

### 📌 HTML 视频说明

**❌ `.html` 不能作为视频源使用。** Qt 的 `Video` 组件只接受**多媒体容器文件**。

如果你有网页上的视频（如 B 站），你需要：
1. 下载为 MP4/WebM 格式，或者
2. 使用 `ffmpeg` 转换：`ffmpeg -i 输入文件 -c:v libx264 -crf 18 输出.mp4`

### 📌 视频规格建议

```
用途            推荐格式    分辨率        码率         帧率
───────────────────────────────────────────────────────────
Resume背景      WebM VP9   2560×1440    ~5 Mbps      30fps
Options背景     MP4 H.264  1280×720     ~3 Mbps      30fps
电源过渡        MP4 H.264  1920×1080    ~10 Mbps     30fps
电源循环        WebM VP9   1920×1080    ~5 Mbps      30fps
```

## 三、🎨 着色器格式

| 格式 | 扩展名 | 支持 | 说明 |
|------|--------|------|------|
| **GLSL 片元着色器** | `.frag` | ✅ **推荐** | 标准 GLSL 源码 |
| **GLSL 通用** | `.glsl` | ✅ 支持 | 与 .frag 等价 |
| **GLSL 顶点着色器** | `.vert` | ✅ **必须配合 .frag** | 壁纸引擎需要 |
| **QSB 编译格式** | `.qsb` | ✅ **推荐** | Qt 预编译，启动更快 |
| HLSL | `.hlsl` | ❌ 不支持 | DirectX 专属 |
| SPIR-V | `.spv` | ❌ 不支持 | Vulkan 格式，Qt 不直接加载 |

### 着色器生成命令

```bash
# 将 .frag + .vert 编译为 .qsb
qsb --glsl "310es,300,100" --hlsl 50 --msl 12 -o output.frag.qsb input.frag
```

## 四、🔤 字体格式（`FontLoader` / `font.family`）

| 格式 | 扩展名 | 支持 | 说明 |
|------|--------|------|------|
| **TrueType** | `.ttf` | ✅ **推荐** | 最通用的字体格式 |
| **OpenType** | `.otf` | ✅ **推荐** | 功能比 TTF 更丰富 |
| WOFF | `.woff` | ✅ 支持 | Web 字体，Qt 可加载 |
| WOFF2 | `.woff2` | ✅ 支持 | 压缩比最高 |
| Type1 | `.pfa` / `.pfb` | ⚠️ 有限支持 | 老旧格式 |
| 位图字体 | `.fon` | ⚠️ 有限支持 | Windows 格式 |

### ⚠️ 注意
- **OTF 和 TTF 都可以用**，`FontLoader` 的 `source` 路径必须写对扩展名
- 字体文件名中的扩展名必须与实际文件匹配（`.ttf` ≠ `.otf`）

## 五、🔄 格式替换建议速查

| 原格式 | 可替换为 | 适用场景 |
|--------|---------|----------|
| `.png` | `.svg` | **图标、Logo、装饰性元素** |
| `.png` | `.webp` | 壁纸、背景（体积更小）|
| `.png` | `.jpg` | 照片类背景（无透明需求时）|
| `.jpg` | `.png` | 需要透明通道时 |
| `.jpg` | `.webp` | 照片类（体积小 30%）|
| `.mp4` | `.webm` | 视频（开源，质量好）|
| `.mp4` | `.mov` | 视频（Apple 生态）|
| `.mp4` | `.mkv` | 视频（多音轨/字幕）|
| `.mp4` | `.html` | ❌ **不能替换** |
| `.frag` | `.qsb` | 预编译着色器（启动更快）|
| `.ttf` | `.otf` | 字体（功能更多）|
| `.ttf` | `.woff2` | 字体（压缩比高）|

## 六、⚠️ 重要限制

1. **❌ HTML 不能替代视频** — `Video` 组件只认多媒体容器格式
2. **❌ GIF 不支持动画** — 只显示第一帧。如需 GIF 效果，用 APNG 或转为逐帧 PNG 序列
3. **⚠️ SVG 不适合大图** — 3840×2194 的 SVG 壁纸会严重卡顿
4. **⚠️ 格式转换建议命令行**：
   ```bash
   # 视频 MP4 → WebM
   ffmpeg -i input.mp4 -c:v libvpx-vp9 -crf 30 -b:v 0 output.webm

   # 图片 PNG → WebP
   cwebp -q 90 input.png -o output.webp

   # 图片 SVG → PNG
   inkscape -w 3840 -h 2194 input.svg -o output.png
   ```
5. **命名空间必须一致** — 替换格式后，文件名（不含扩展名）最好保持不变，否则需要修改 QML 代码中的引用路径

## 七、最适合 BA 主题的格式组合

```
壁纸/背景   → PNG 或 WebP（无损+透明支持）
角色立绘    → PNG 或 JPEG（照片类）
图标        → SVG（可无限缩放）
UI边框      → SVG 或 PNG
视频背景    → WebM VP9（高质量+小体积）
电源过渡    → MP4 H.264（兼容性最佳）
字体        → OTF（功能丰富）
着色器      → GLSL .frag + .vert 或 QSB 预编译
```
